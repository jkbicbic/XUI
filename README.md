# XUI (Incomplete)

This is an incomplete Experimental CSS Framework for WIKIX project that's not yet complete. My goal is to make this framework a standard for all incoming projects (hopefully).

### Framework Structure
```
projectFolder
  |--scss
  |  |--button.scss
  |  |--card.scss
  |  |--colors.scss
  |  |--common.scss
  |  |--footer.scss
  |  |--global.scss
  |  |--grid.scss
  |  |--img.scss
  |  |--input.scss
  |  |--main.scss
  |  |--nav.scss
  |  |--responsive.scss
  |  |--section.scss
  |  |--text.scss
  |--app.scss
```

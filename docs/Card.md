# Card
----
apply the `card` class to a `<div>` tag

```HTML
<div class="card">
    <div class="card__header">
        ...
    </div>
    <div class="card__body">
        ...
    </div>
    <div class="card__footer">
        ...
    </div>
</div>
```

#### Card Images

If you want images to be displayed just add the snippet below indside the `<div class="card">`.

```HTML
<div class="card__img">
    <img src=""/>
</div>
```

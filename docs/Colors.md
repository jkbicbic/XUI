# Colors
---
I'm planning to centralize each color in a single `scss` located in `/scss/colors.scss`

example
```CSS
/* Card Color */

$card-bg: #hex;
$card-header-text: #hex;
$card-text: #hex

/* Button */

$btn-default: #hex;
$btn-text: #hex;

```
